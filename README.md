FactuScan - Analyse Intelligente des Factures Marocaines
FactuScan est une application web d'analyse intelligente des factures marocaines qui automatise l'extraction des informations via l'OCR et l'IA, avec un assistant vocal bilingue (Français/Arabe).

Fonctionnalités
Importation de factures (PDF, JPG, PNG)
OCR bilingue (Français/Arabe)
Extraction intelligente des informations clés:
Numéro de facture
Date
Montant TVA
Montant total
Fournisseur
ICE (Identifiant Commun de l'Entreprise)
Interface utilisateur épurée
Assistant vocal bilingue (Français/Arabe)
Exportation des données
Historique des factures
